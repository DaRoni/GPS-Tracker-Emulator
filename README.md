# GPS-Tracker-Emulator
Простенький эмулятор GPS-tracker (em_gps) работающий через COM-port.

Требования:\
Требуется эмулятор COM-ports [Null-modem emulator](https://sourceforge.net/projects/com0com/). \
Установить Python 3 с модулем pip.\
Установить библиотеку Python через консоль (Powershell) командой -m pip install pyserial.

Инструкция:\
Устанавливаем программу виртуальных COM портов com0com.\
В диспетчере устройств появится 2 новых COM порта.\
Один из COM портов для передачи данных, а другой для получения данных координат.\
В файле gps.txt исходные данные.\
В первой строке указываем виртуальный COM-port программы com0com для эмулирования GPS трекера.\
В строках ниже указываем координаты, последние две цифры координат случайные для погрещности.

Координаты берём из YandexMaps (https://yandex.kz/maps/ru)  и записываем с точкой в файл gps.txt.\
COM4\
50.529947\
83.417209

В настройках программы где требуются GPS трекер, указываем другой COM порт для получения данных координат.
